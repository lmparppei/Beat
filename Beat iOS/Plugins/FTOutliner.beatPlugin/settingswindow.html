    Production Helpers 
  • When ON, a note or marker with a percentage such as [[50%]] anywhere in a scene will make it display in red to green based on the value. Scenes with no percentage will show as white.
    Toggle this by typing % to temporarily switch to see what scenes remain to finish.


    Behavior Settings 
  • When on, use cmd+click to override this and keep focus in FTOutliner.
    cmd+click on empty space to avoid/reset selecting multiple.
  • When OFF, the same can be achieved ad hoc with ctrl+alt+cmd+click.
  • When OFF, markers don't display text in non-proportional and index card mode. When ON, marker text is visible, but markers may obscure each other as well as the content of index cards. Use wisely!
